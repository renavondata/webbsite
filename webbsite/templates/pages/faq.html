Frequently Asked Questions

General Questions

What is the Renavon Hong Kong Database?

The Renavon Hong Kong Database is a comprehensive archive of Hong Kong financial market data spanning over 35 years (1990-2025). It contains information about listed companies, directors, beneficial ownership, corporate actions, SFC licensees, solicitors, and much more.

Is this data free to use?

Yes! The database was released under the Creative Commons CC-BY 4.0 license in October 2025. You are free to use it for commercial or non-commercial purposes, as long as you attribute Webb-site.com as the source.

What is the relationship with Webb-site.com?

This database contains data originally compiled by Webb-site.com, founded and operated by David M. Webb MBE since 1998. David generously open-sourced the database schemas and scrapers in October 2025 to preserve public access to this valuable Hong Kong financial data.

Are Webb-site articles included?

No. The editorial content from Webb-site Reports (articles, analysis, opinions) remains copyright David M. Webb and is NOT included in this database. Where articles are referenced, we link to Archive.org preserved copies.

Data Coverage

What time period does the data cover?

  • Stock Quotes: Daily since 1994-01-03
  • CCASS Holdings: Daily since 2007-06-26
  • Corporate Actions & Directors: Since 1990
  • SFC Licensees: Since 2003-04-01
  • HK Solicitors: Current with historical moves

How often is the data updated?

Different data sources are updated on different schedules:

  • Stock quotes and CCASS: Daily
  • Financial reports: Continuous monitoring
  • SFC licensees & HK solicitors: Daily
  • Companies Registry: Weekly
  • UK Companies House: Continuous via API

What makes CCASS data special?

Our CCASS holdings data goes back to 2007-06-26, while HKEX public data only shows 1 year of history. This allows analysis of long-term beneficial ownership trends. Once a stock is delisted, HKEX deletes the data immediately, making historical CCASS data unrecoverable without independent archiving.

Using the Database

How do I search for a company?

Use the Company Search box at the top of any database page. You can search by:

  • Company name (English or Chinese)
  • Stock code (e.g., "5" for HSBC)
  • Incorporation number

How do I search for a person?

Use the Person Search box at the top of any database page. Enter:

  • Family name in the first box (e.g., "Webb")
  • Given name(s) in the second box (e.g., "David")
  • Check options for exact match vs. full-text search

How do I find CCASS holdings for a stock?

There are several ways:

  • Search for the company, then click on CCASS links in the navigation
  • Use Top CCASS Changes to see recent significant changes
  • Use CCASS Concentration Analysis to analyze ownership concentration

What are "Webb Total Returns"?

Webb Total Returns (WTR) measure stock performance including both capital gains and reinvested dividends. This gives a more accurate picture of investment returns than price changes alone. See About Total Returns for methodology details.

Technical Questions

Can I download the entire database?

The database schemas and scrapers are open source on GitHub. You can set up your own mirror using PostgreSQL. Contact us for database dump files.

Is there an API?

Currently, there is no formal API. However, all pages support direct URL queries (e.g., /dbpub/orgdata.asp?p=16954). You can bookmark or link to specific queries. Please be respectful with automated access.

What database technology is used?

The production system uses:

  • Database: PostgreSQL 16
  • Web Framework: Flask (Python 3.11)
  • Hosting: Render.com cloud platform

Can I contribute code or data?

Yes! The project is open source. Visit our GitHub repository to contribute. We welcome bug fixes, feature enhancements, and data quality improvements.

Data Quality & Corrections

What if I find an error in the data?

Please contact us with details about the error. Include:

  • The page URL where you found the error
  • Description of what's wrong
  • Correct information (if known)
  • Source documentation (if available)

How accurate is the data?

Data is scraped from official sources (HKEX, SFC, Companies Registry, etc.) and generally reflects what those sources publish. However:

  • Source data may contain errors or be outdated
  • Scraping may occasionally miss updates during system maintenance
  • Complex data (like cross-holdings) requires manual verification
